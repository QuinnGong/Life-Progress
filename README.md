# Life Progress

## Features

- **Life Map Visualization**: Displays your entire expected lifespan as a grid of dots (52 weeks per row).
- **Real-time Updates**: All settings take effect immediately without a refresh button.
- **Color Customization**: Choose your own "passed time" color or pick from elegant presets.
- **Transpose Layout**: Switch between portrait (Years as rows) and landscape (Weeks as rows) views.
- **4K Wallpaper Export**: Right-click on the grid to save it as a high-resolution 4K image, perfect for phone or desktop wallpapers.
- **Persistent Settings**: Your birthday, lifespan, and color preferences are saved locally and restored on next visit.
- **Breathing Animation**: The current week dot pulses gently to mark where you are in life.

## Usage

1. Open `index.html` in any modern browser.
2. Set your **Birthday** and **Expected Lifespan**.
3. Customize the color if desired.
4. Right-click the grid to download a 4K wallpaper.
